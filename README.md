# kuwadro
Instant pose capture and event-ready photo layouts for weddings, birthdays, and parties. A lightweight web app that runs pose detection in the browser, auto-captures on gestures like a peace sign, and applies customizable templates so guests leave with polished, shareable images.

Kuwadro is a privacy-first, client-heavy photo booth web app that detects user poses in real time and triggers automatic captures when a configured gesture is held. Captures are rendered into themed templates (wedding, birthday, debut, fiesta) and can be downloaded or uploaded to a storage backend. The app is optimized for free-tier hosting and on-device inference to minimize cost and latency.
